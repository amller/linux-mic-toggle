# alsa-mic-toggle
## Dependcies
The following packages need to be installed:
- alsa-utils

## GNOME Setup
1. Open a Terminal and run `git clone https://github.com/amller/alsa-mic-toggle`.
2. Open Settings and go to Keyboard -> Keyboard Shortcuts -> View and Customize Shortcuts -> Custom Shortcuts
3. Create a new Shortcut which executes the following command: `sh <absolute path to cloned repo>/mic-toggle.sh`
